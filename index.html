Sublime 入门指南

Sublime 入门指南

这两天一直在折腾 Sublime 编辑器，从听都没听过到这几天瞎折腾，终于对这款装逼神器有了大致的了解。

简介

Sublime Text：一款具有代码高亮、语法提示、自动完成且反应快速的编辑器软件。编辑器有很多，古老的有 VIM，emacs 这样的神之编辑器，新生的有 sublime，github 自己开发的 Atom，也有台湾开发的 Notepad++。 那 sublime 有什么吸引我们的地方呢？我觉得最大的优势是

  • 丰富插件
  • 体积轻巧，反应神速
  • 可以 DIY 适合自己的主题，geek 范十足

插件

插件是 sublime 最强大的功能了，你可以把一个轻巧的编辑器弄成媲美于 Eclipse 的 IDE。插件安装推荐使用 Package Controll 组件安装，Package Controll用来管理插件也很方便。

按 ctrl + `(tab上面那个键)调出 console 输入并回车运行：

重启Sublime Text 3。如果在Perferences->package settings中看到package control这一项，则安装成功。按下 Ctrl+Shift+P 调出命令面板输入install 调出
Install Package 选项并回车，然后在列表中选中要安装的插件。

主要的插件：sublimeGit（sublime 3 才有的），sublimeLinter, Side Bar, GitGutter，MarkDown Editing 可以一个一个去了解下。

MarkDown Editing

SublimeText不仅仅是能够查看和编辑 Markdown 文件，但它会视它们为格式很糟糕的纯文本。这个插件通过适当的颜色高亮和其它功能来更好地完成这些任务。
使用方法：按下 Ctrl+Shift+P 安装插件，在 Ctrl+Shift+P 里面 set syntax:markdown

Sublime Terminal

这个插件可以让你在Sublime中直接使用终端打开你的项目文件夹，并支持使用快捷键。

SublimeLinter

SublimeLinter 是前端编码利器——Sublime Text 的一款插件，用于高亮提示用户编写的代码中存在的不规范和错误的写法，支持 JavaScript、CSS、HTML、Java、PHP、Python、Ruby 等十多种开发语言。这篇文章介绍如何在 Windows 中配置 SublimeLinter 进行 JS & CSS 校验。

SideBarEnhancements

SideBarEnhancements是一款很实用的右键菜单增强插件；在安装该插件前，在Sublime Text左侧FOLDERS栏中点击右键，只有寥寥几个简单的功能；安装了就相当于给其丰了大胸一般。 更强大的是，该插件还能让我们自定义快捷键呼出某个浏览器以预览页面！这样就不用到项目目录下寻找和拖动到特定浏览器中预览了。

快捷键

  • Ctrl+Shift+L 将当前选中区域打散，然后进行同时编辑（黑客感十足）
  • Ctrl+J 把当前选中区域合并为一行
  • Ctrl + P 匹配到文件
  • Ctrl+Shift+P 最需要用到的快捷键
  • Ctrl + D 选中一个变量，按此快捷键可以同时编辑相同变量。

定制属于自己的个性化主题

SublimeText 她不仅仅是作为性感无比，强大可大编辑器的存在；要知道在如今这么靠脸吃饭的时代，她的美者这方面怎么可能不臻至完美呢？Mac和Windows都默认十多种不同风格的主题，想必会有您喜欢😍的。即便没有也没关系，我们可以自己定制自己喜欢的风格主题。在线制作专属Sublime Text主题。这个网址可以轻松按照自己的意愿制作喜欢的主题。完毕之后将生成的xx.sublime.theme文件，点开Preferences -> Browsr Packages这个，放在这个直属目录之下，即可在Theme处选择这个主题了。

一点感触

这两天都在折腾 Sublime ，一个插件一个插件了解过去，sublime 确实强大，现在还有个 Atom 开始火起来，大有赶超 Sublime 趋势，有空可以去了解下。工欲善其事，必先利其器，要想上山砍柴，先磨好自己的刀吧。但是：

工匠应该专注于作品的创意，不应该浪费精力，没限制地在折腾自己的工具。

所有也不能完全沉溺在工具的对比折腾中，正如论语所说的

虽小道，必有可观者焉；致远恐泥，是以君子不为也。

夸父逐日

第一章
沼泽村的鸡总是比其他地方的要早活动，才两点就有几只不安分的在那里打起鸣来。这让躺在床上刚去睡没多久的夸父清醒了起来，他转头望了下挂在正对门口大厅上的大摆钟，时针和分针摆成一个小角畏缩在数字4身上。家里的大摆钟放在那边墙上的年月可不比夸父年龄小，那一年夸父刚出生还没周岁，夸父的父亲挑着一担茶去集市换了一猪脚给孩子抓阄用还有就是这一个大摆钟和一箱白干面，近几年时针已经有些脱落会早个一小时。夸父猜了下时间才三点多，但此时已睡意全无，任由身体躺在床上，眼睛漫无目的地巡视着屋顶。这屋顶由黑瓦片一倒排接着一顺排用来引雨水，只有中间一小口用玻璃代替当做小天窗，月光从透明的玻璃面倾泻下来，在房间内流转，云气氤氲。这里的一切环境既让夸父熟悉又感到厌烦，他拿手按了按太阳穴用力摇了摇头，随即坐了起来。周围还是静悄悄的，只有不知名的虫鸣声和偶尔远处传来的鸡鸣声，除了照在门口地板上的月光，一切都在黑暗的笼罩中。突然，夸父发现了大厅有闪过几道光影，静声细听还有木柴烧旺的霹雳声，便轻手轻脚的穿上衣服走了出来。
厨房里，母亲林氏正弯下腰拾起烧到一半掉在地上的木枝往灶台扔，再往里面放上一根较大的柴木。夸父望着她的身影轻喊了一声“阿姆”。林氏回头一看，笑着说：“怎么这么早？你可以去多睡一会儿，等到五点我再叫你，咱们村的汽车可是六点才走的。”
“你怎么这么早起来？”
“我起来煮碗粥给你喝，起早赶路不喝碗热身对身体不好。”随即转身拿起手电筒往灶台一照，看了下粥熟了没。
夸父注意到厨房的电灯已经坏掉，看了下生锈的电灯盖，估计坏了好几年年头了。以前总是忽略，突然的发现令他感到羞愧。
“我帮你把厨房的电灯换一下。”
“不用，都坏了五六年，我都习惯了，不用你多费劳。”
但是夸父心想自己即将离开，执意要换，便去找梯子和灯泡。林氏是一个很固执的人，怨声说道：“都说了不用换了，我是个习惯黑暗的人，不像你还想着挑高枝去追太阳。”随即驼着背拿起鸡饲料往后门去喂养家禽。夸父就当没听见，把梯头放在墙角，人爬了上去，想用螺丝刀把电灯盒子打开，但是已经生锈得太厉害了，他只好拿起剪刀剪掉，用打火机烧了下电线，然后趁热用手指把外面的那层橡胶皮拔掉，换上了一个新的电灯盒盖，再安上新的六十瓦的钨丝灯泡。忙活了半个小时，终于把厨房的灯泡给点亮了。夸父开心的望了下屋顶红色光芒的灯泡，想等母亲进来夸奖下他，过了一会儿，林氏拿着装鸡饲料的桶进来，却用另一只手挡着眼睛，似乎光线太亮令她不适应。夸父脸上愉快的表情渐渐沉了下去，但内心里要离开的念头变得更加坚定。他想着还有些事情要交代大哥，便往后门拐弯走了上去。
大哥正在水池旁磨着柴刀，而大嫂在一旁洗着衣服，两个人都没说话，各做着各自的事。夸父一上去，大嫂就问：“小弟你是今天要去是吧？吃了没，在上面吃，我煮了你的份。”
“不用了，阿姆下面快煮好了。”
大嫂感叹道：“还是像你这样有前途，出去打拼，万一追到了，一辈子吃喝不愁。”
夸父撇着嘴唇轻笑着，没回答，转头向着大哥“阿姆就烦你先照顾，那些田地你先拿去做，做不了就给别人些。”
“祖辈留下来的，怎么舍得让给别人。”大哥正声说着。舀起一盆水往打磨石上撒了撒，继续磨着那边柴刀，他们准备着今天去蔡尖尾山砍柴。
夸父也没在说什么，静静地望着门对面的蔡尖尾山，此时天才刚亮，尖尾山笼罩在一层大雾里，那大雾从山顶倾泻下来，直到家门口，整个沼泽村说不出的白茫茫一片，大哥和嫂子各自做着自己的事，空气中只留下磨刀时来回有节奏的哗哗声。
大哥停下磨刀活，若有所思的说道“有些事按理不该说，但你要走了，也不知什么时候回来，顺便跟你说下吧。”
“什么，你说。”
“你也都快三开头了，不成家也不是事儿，而且这种事急不来。要不先把老二过继给你，好让你少些忧虑，多些盼头。”
夸父冷笑着，没有说什么。大嫂随即迎合着“对啊，小叔，老二可乖巧，入了你的明烧上黄金纸，这在外人看来才像样嘛！”
夸父望着大哥和大嫂，愤愤的说声：“你还不是惦记着那几份田头地，把整个沼泽村都给你你做的来嘛？”
大嫂一副申冤的表情“小叔你这怎么说话，我和你大哥还不是为了你好，你想想当初阿爸走的时候你才几岁，还不是你大哥和我把你养大的，你虽然和阿姆一起吃，但农活我们可是一起做，出的工比你们多了多少。现倒好了养了个白眼狼，还说我们惦记着你的田头地，这些田头地就算我们拿来了又如何，有些本来就是我们用心得来的。”大哥虽然没说话，但是也面露怒气。夸父没再说什么，转头走了下去，他发誓一定要离开这个鬼地方。

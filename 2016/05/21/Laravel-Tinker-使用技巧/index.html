Laravel tinker 使用技巧

Tinker 是上面

Tinker 是 Laravel 自带的一个命令行的 PHP 交互工具。
Tinker只是在laravel中的名字，其实真名叫做psysh。

显示执行的 mysql 语句

tinker是一款可以在命令行对数据库进行操作的强大工具，比如我们可以执行$post = App\Models\Admin::first();
命令行会直接显示出数据库 admin 表里面第一行的字段。

在命令行中执行 DB::listen(function($query) { var_dump($query->sql); }); 再执行一次上面的命令，就能显示执行的 mysql 语句。
